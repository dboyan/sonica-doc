# Roadmap for Future Releases

## Planned features for Q1 2023

* eNodeB MAC layer:
  - Configurable MCS for uplink and downlink transmissions
  - UL retransmission
  - DL HARQ handling
* eNodeB PHY layer:
  - Support for NPUSCH Format 2
* EPC improvement:
  - Improve message support for more devices
